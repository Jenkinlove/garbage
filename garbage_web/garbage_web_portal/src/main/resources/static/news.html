我国不同城市之间对于不同垃圾分类方法的比较1
2019-11-07 22:33:17

垃圾分类的热潮已经过去将近两个月了，但是“垃圾分类”依然是热度不减，因为现在全国各地几乎都在落实垃圾分类处理这一政策。那么问题来了，由于我国各个地区的对于垃圾分类的标准不是完全一致，所以就导致了我国现在正在实施垃圾分类的46个重点城市的垃圾分类方法有些不同，甚至有的城市根本就不像上海一样存在干湿垃圾的分类标准，下面就由垃圾爱分类小编给大家介绍一下我国各个地区对于垃圾分类的不同标准吧 我国各个地方城市对于垃圾分类的不同政策要点 从这两个月实施垃圾分类的城市来看，现阶段我国各个城市对于垃圾分类的标准一共有五种之多，当然这还只是现阶段，以后说不定会不会有变化。这五种垃圾分类的标准又可分为三种分法，下面就由垃圾爱分类小编具体分享一下： 一、三分法 三分法是指将我们日常生活中所产生的垃圾分为三种，分别是： 可回收物、有害垃圾、其他垃圾。三分法这种垃圾分类的标准目前只在我国非常小一部分城市使用，如：广元、咸阳、拉萨等城市。 二、四分法 四分法是指将我们日常生活中所产生的垃圾分为四种，这种四分法是现在我国各个城市对于垃圾分类使用最为多的一种标准，如：上海、北京、重庆、哈尔滨、杭州等经济发展较好的城市。不过值得一提的是，四分法标准在这些城市当中又被分为了三个小派系，分别是： 易腐垃圾、可回收物、有害垃圾、其他垃圾。目前正在根据这种垃圾分类标准的城市有：重庆、石家庄、太原、大连、长春、苏州、杭州、铜陵、南昌、海口、德阳、贵阳、昆明、兰州等城市。 餐厨垃圾、可回收物、有害垃圾、其他垃圾。目前正在根据这种垃圾分类标准的城市有：北京、呼和浩特、沈阳、哈尔滨、南京、宁波、合肥、厦门、广州、宜昌、青岛、泰安、济南、郑州、宜春、南宁、成都、乌鲁木齐、西宁、西安、银川等城市。 湿垃圾、可回收物、有害垃圾、干垃圾。目前正在根据这种垃圾分类标准的城市有：上海、邯郸等城市（更多垃圾分类城市）。 三、五分法 五分法是垃圾分类方法当中最为特殊的一种方法，为什么说它特殊呢？因为在五分法当中有一种垃圾是我们没有听到的过的，那么五分法具体分法就是将垃圾分成五种，分别是：厨余垃圾、可回收物、有害垃圾、其他垃圾、大件垃圾。 对于“大件垃圾”不了解朋友没有关系，也不必太过于纠结，目前据垃圾爱分类小编所知，这种垃圾分类的五分法只在我国的福州市的垃圾分类政策当中。那么在福州的小伙伴要注意了，由于福州垃圾分类的方法与其它城市的垃圾分类方法是不一致的，所以千万千万不要被误导了哦！
